# Roadmap

This document outlines potential future features and improvements. Community input welcome!

## Core Features ✅
- [x] Dynamic passage loading via fetch API
- [x] Markdown rendering with marked.js
- [x] Custom link syntax (direct, relative, random groups)
- [x] Transition system with automatic fallback
- [x] Animation macros (wiggle, typewriter, fadein, delayed)
- [x] Manifest generation script
- [x] Passage preloading
